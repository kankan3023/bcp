# BCP自動生成スキル

企業WebサイトのURLを入力するだけで、**BCP（事業継続計画）** を自動生成するClaude Codeスキルです。

## 特徴

- **ワンコマンド実行**: URLを渡すだけで自律的にBCPを生成
- **実データに基づくリスク分析**: 国土交通省ハザードマップポータルのタイルデータを使用
- **業種×地域に特化**: 汎用テンプレートではなく、企業固有のリスクと対策を提示
- **中小企業庁ガイドライン準拠**: BCPのセクション構成は公式指針に準拠
- **PDF出力**: A4印刷対応のプロフェッショナルなPDFを生成

## 使い方

### 前提条件

- [Claude Code](https://claude.ai/code) がインストール済み
- Python 3.10以上

### インストール

```bash
pip install -r plugins/bcp-generator/requirements.txt
```

### 実行

Claude Code上で以下を実行:

```
/bcp https://example-company.co.jp
```

### 出力

`/tmp/BCP_{会社名}_{日付}.pdf` にBCP文書が生成されます。

## BCPの構成

生成されるBCPは以下のセクションで構成されます:

1. **表紙** - 会社名、作成日
2. **目次**
3. **会社概要・事業概要** - Webサイトから自動収集
4. **BCP基本方針** - 目的、適用範囲
5. **リスク分析** - ハザードマップデータ + 地震想定 + 業種固有リスク
6. **重要業務の特定** - AI分析による業務優先順位
7. **目標復旧時間（RTO）** - 業種×リスクに基づく設定
8. **対策・行動計画** - 事前/発災直後/復旧の3フェーズ
9. **緊急連絡体制** - 記入欄付きテンプレート
10. **備蓄・資源計画** - チェックリスト
11. **付録** - データ出典、免責事項

## 技術構成

| コンポーネント | 技術 |
|-------------|------|
| ジオコーディング | 国土地理院API + Nominatim（OSM） |
| ハザード分析 | 国土交通省ハザードマップタイル（洪水・土砂・津波） |
| 地震情報 | WebSearch による自治体防災情報収集 |
| PDF生成 | WeasyPrint（HTML/CSS → PDF） |
| AI分析 | Claude による業種×地域リスク統合判断 |

## 自律判断ポイント

このスキルは3つの場面でAIが自律的に判断を行います:

1. **リスク優先順位の決定** - ハザードデータ×業種から企業固有のリスクTOP3を特定
2. **重要業務とRTOの決定** - 業種特性と脆弱性を考慮した復旧目標の設定
3. **BCP文書生成** - 具体的な対策を含む完全なBCP文書の作成
